TigerNET
=============

A Tiger compiler that generates IL developed in C#

About this project
-------

This is an academic project which consists on developing a Tiger compiler in C#

### Tools/languages used:

* `ANTLR` (Lexer & Parser)
* `C#` (Semantic check & Language generation)
